a docker container to run Quake II demo game on a linux machine

to run execute:
```
git clone git@github.com:ivanaslamov/quake2-docker-linux-intel-gpu.git
cd quake2-docker-linux-intel-gpu
docker build -t quake2 .
docker-compose up
```
